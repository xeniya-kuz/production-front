## Фича выставления оценки для статьи

Описание: фича для оценки статьи пользователем. Загружает текущий рейтинг статьи и позволяет выставить новую оценку с опциональным текстовым отзывом.

#### Public api

- Components

`ArticleRating` - Компонент оценки статьи со звёздным рейтингом и формой обратной связи
