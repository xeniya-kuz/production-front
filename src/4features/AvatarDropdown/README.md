## Фича для дропдауна под аватаркой

Описание: фича выпадающего меню пользователя. Открывается по клику на аватар и содержит ссылки на профиль, настройки, панель администратора (для администраторов/менеджеров) и кнопку выхода.

#### Public api

- Components

`AvatarDropdown` - Выпадающее меню с аватаром и навигацией пользователя
