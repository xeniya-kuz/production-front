## Фича для авторизации по имени пользователя

Описание: фича авторизации пользователя по логину и паролю. Включает модальное окно с формой входа и управление состоянием авторизации.

#### Public api

- Components

`LoginModal` - Модальное окно с формой авторизации

- types

`LoginSchema` - Тип состояния Redux (username, password, isLoading, error)

- reducers

`loginReducer` - Редьюсер для управления состоянием формы авторизации
