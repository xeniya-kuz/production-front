## Фича смены отображения списка: список или плитка

Описание: фича переключения режима отображения списка статей — плитка (TILE) или список (LIST). Сохраняет выбранный режим в пользовательских настройках и обновляет лимит загружаемых статей.

#### Public api

- Components

`ViewSwitcher` - Переключатель режима отображения статей (список/плитка)
