## Сущность комментария

Описание: сущность комментария к статье. Содержит информацию о тексте комментария и авторе.

#### Public api

- Components

`CommentList` - Компонент для отображения списка комментариев с поддержкой состояния загрузки

- types

`Comment` - Тип, описывающий комментарий (id, text, user)
