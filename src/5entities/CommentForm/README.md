## Сущность формы комментария

Описание: сущность формы для добавления комментария. Управляет состоянием текстового поля и процессом отправки.

#### Public api

- Components

`CommentForm` - Форма ввода комментария с кнопкой отправки

- types

`CommentFormSchema` - Тип, описывающий состояние формы комментария (comment, error, isLoading)

- reducers

`commentFormReducer` - Редьюсер для управления состоянием формы комментария
