## Сущность рейтинга

Описание: сущность оценки (рейтинга). Предоставляет компонент для выставления звёздного рейтинга с возможностью оставить текстовый отзыв.

#### Public api

- Components

`RatingCard` - Карточка оценки со звёздным рейтингом и опциональной формой обратной связи

- types

`Rating` - Тип, описывающий оценку (rate 1–5, feedback)
