## Виджет редактирования профиля пользователя

Описание: виджет полноценной карточки профиля с возможностью просмотра и редактирования. Включает заголовок с кнопками управления (редактировать/сохранить/отмена), отображение ошибок валидации и асинхронную загрузку редьюсера профиля.

#### Public api

- Components

`EditableProfileCard` - Карточка профиля с режимами просмотра и редактирования
